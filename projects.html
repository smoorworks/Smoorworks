finished projects:

The Tree And The Axe

This a game about chopping down trees. This is our first ever game and we made it to get comfortable with game development. So don't expect it to be the best game ever.

Unsolved Memories

This is a psycological horror game where you walk through a dark hallway and uncover the story of the main character.
This project is made for the GMTK Game Jam 2025 with the theme "loop", this game fits the theme because you loop in the same hallway throughout the game.

Death Light

This is a small puzzle like game. You need to find a way trough the shadows to get to the door on the other side of the map.
This project is made for the Dutch Game Jam with the theme "Shadow", this game fits the theme because you need to walk in the shadow to get to the other side.
